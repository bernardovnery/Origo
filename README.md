# ORIGO — Manual de Marca

Manual oficial de identidade visual da **ORIGO** — infraestrutura institucional de tokenização para o agronegócio brasileiro.

> *Originação para a economia tokenizada.*

---

## 🌐 Site

Versão publicada: [https://SEU-USUARIO.github.io/origo-manual](https://SEU-USUARIO.github.io/origo-manual)

## 📐 Conteúdo

Este manual estabelece os padrões oficiais de uso da marca ORIGO:

- **Identidade visual** — essência conceitual e direção estética
- **Logotipo** — versões primária, reversa, vertical e ícone isolado
- **Sistema cromático** — paleta institucional com valores RGB, CMYK e PMS
- **Tipografia** — Cormorant Garamond, Inter e JetBrains Mono
- **Aplicações** — papelaria executiva, apresentações e mídia digital
- **Usos incorretos** — diretrizes do que evitar
- **Central de downloads** — arquivos SVG vetoriais

## 🎨 Paleta institucional

| Cor | HEX | Uso |
|---|---|---|
| Navy Institucional | `#0D1F3C` | Cor primária · textos · superfícies escuras |
| Dourado Origo | `#C8960C` | Acento · ícone · destaques |
| Marfim | `#F2EFE8` | Backgrounds · superfícies neutras |
| Papel | `#FFFFFF` | Fundo institucional |
